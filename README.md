# 팀명: GrayCorn
### 🔗 [프로젝트 요약](https://rollslois.notion.site/1ecf3ba1b3c844f1a04b31e8e57cacb7)
- 팀원
  - 김륜경, 장우성, 정계진, 조혜원, 채근영
- 프로젝트 주제
  - 배달 플랫폼의 음식점 리뷰를 분석하여 사용자가 등록한 별점과 리뷰의 실제 내용의 일치 여부를 판별하여 고객에게 음식점에 대한 다양한 인사이트 제공
